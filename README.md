## Body Mass Index Calculator
### Technologies Used:
#### HTML5, CSS3, React.js
 - $ :  `git clone https://github.com/getwebem/bodyMass.git`
 - $   :  `cd bodyMass`
 - $  :  `npm install`
 - $  :  `npm start`
 - Web Browser :  `http://localhost:3000`
