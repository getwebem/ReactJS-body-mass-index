## Body Mass Index
#### :rocket: HTML5, CSS3, React.js
 - $ : `git clone https://github.com/getwebem/ReactJS-body-mass-index.git`
 - $ : `cd ReactJS-body-mass-index`
 - $ : `npm install`
 - $ : `npm start`
 - Web Browser :  `http://localhost:3000`
